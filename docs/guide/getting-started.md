## 前置条件

- [vitepress](https://vitepress.dev/) 1.3.2及以上版本
- 完成 [所有步骤](https://vitepress.dev/zh/guide/getting-started#installation) 并且成功启动运行

## 安装

::: code-group

```sh [pnpm]
pnpm add @theo-messi/tm-fe
```

```sh [npm]
npm install @theo-messi/tm-fe
```

```sh [yarn]
yarn add @theo-messi/tm-fe
```

:::
